# 🚀 Quick Setup - Workflow Hub GUI

**Get your executive command center running in 3 simple steps!**

## Step 1: Setup Dependencies 
```bash
cd ~/Desktop/workflow-hub
python3 setup_venv.py
```
*This creates a virtual environment and installs all required Python packages*

## Step 2: Create macOS App
```bash
python3 create_macos_app.py
```
*This creates a double-click .app bundle for your desktop*

## Step 3: Launch! 
**Double-click "Workflow Hub.app" on your desktop** 
*Or run manually:*
```bash
python3 start_gui_with_venv.py
```

## 🆘 Troubleshooting:

**"Swift not found":**
```bash
xcode-select --install
```

**"Dependencies missing":**
```bash
python3 setup_venv.py
```

**"App won't start":**
Check that both commands above completed successfully.
