## 🌅 Daily Workflow (Start Here!)

### 1. Morning Setup (5 minutes)
```bash

```

This will:
- Scan all your active projects
- Generate optimized task assignments
- Create parallel execution plan
- Set daily productivity targets

### 2. Work Execution (Throughout Day)

This ensures all context and learnings are preserved for tomorrow.

## 🚨 Troubleshooting

### Common Issues

#### "Agents not responding"
```bash
# Check MCP connection
python orchestration/agent_mcp_bridge.py --status

# Restart Context Engineering MCP
~/.local/bin/start-context
```

#### "Context not persisting"
```bash
# Repair context connections
python orchestration/context_repair.py --fix-all

# Check context field status
python orchestration/agent_mcp_bridge.py --context-status
```

#### "Performance degrading"
```bash
# Rebalance agent workloads
python orchestration/load_balancer.py --rebalance

# Clear performance history and restart
python metrics/productivity_tracker.py --reset
```
