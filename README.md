## 🚀 Quick Start

### 1. Initialize a New Project
```bash
cd workflow-hub
python orchestration/project_init.py --project "New Venture" --type "business"
```

### 2. Execute a Workflow
```bash
python orchestration/workflow_runner.py --workflow "business_development" --project "Ocean Paradise"
```

### 3. Daily Productivity Session
```bash
# Morning - Review and plan
python daily-ops/morning_review.py

# During work - Execute with agents
python orchestration/agent_coordinator.py --mode parallel

# Evening - Persist learnings
python daily-ops/evening_persistence.py
```
